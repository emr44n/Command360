The branding line. The post format — avatar, handle, caption, like/comment counts — is functional and fine. Real logos, wordmarks and exact trade dress are the risk, and clone-branded assets are a procurement red flag for police/fire/gov buyers. So the default ships generic-but-legible ("PhotoShare", "Chirp", "News 24"). Fidelity to any real platform is a dial you control — but generic is the safe, better-ageing choice. (Not legal advice — worth an IP check at launch.)

The data model you built toward

Each View is a slide — and its own ecosystem.

You already think of it as a presentation where every slide is a full incident. Formalised: a Scenario holds Views; each View carries its own layers, its own Phases → Injects → Steps, and its own hotspots. Cutting slides = cutting views.

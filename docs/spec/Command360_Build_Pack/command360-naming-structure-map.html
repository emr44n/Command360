Why "Command" leads everything: it's the industry's own root word. ETC's ADMS sells Incident Command, Command Post and Command Simulation; XVR and Hydra use the same register. "Command Classroom / Command Live" belongs in that lexicon — you're speaking their language, not inventing one.

The two units

A TDE for the classroom. A Scenario for live.

Both names are the industry's own — and deliberately not "presentation," which invites the "just a PowerPoint" perception. Everything runs as an assessment; there's no separate training/assessment split.

Command Classroom

TDE — Tactical Decision Exercise

The recognised term (Effective Command, Hydra, College of Policing). Assembled from blocks: polls, quizzes, word clouds, Q&A, surveys, content, rating scales, open text. The blocks stay generically named — they're the materials, not the unit.

Command Live

Scenario

The industry default (XVR, Fire Studio, ADMS). Assembled from Phases → Injects → Steps, using layers built in Command Studio. Optionally labelled by level — Bronze (operational) / Silver (tactical) / Gold (strategic) — for categorising.

What we took from the XVR manual

The same logic, renamed to be ours.

XVR On Scene is entirely event-driven: the operator fires Events made of Actions; linked actions play together, unlinked in sequence, a Wait inserts a gap. We keep the logic and rename almost everything.

Inside a Scenario

Phase → Inject → Step.

Steps run top-to-bottom. Link them to play together; leave them unlinked to run in order; drop a Wait for a gap. That single rule choreographs an incident.

Beyond XVR

The bits that make it 2026, not a 3D simulator.

Multi-angle

Views

Front · rear · side · from the cab. Switchable plates the instructor cuts between when the candidate moves — "I want the rear."

Phone companion

Second screen

An on-screen QR opens a page on the candidate's phone (no app). Push maps, cards and 999 footage live — session-scoped, expiring, watermarked.

Live filters

Looks & FX

Real-time colour, contrast, blur, thermal — no re-render. AI generates structural variants (damaged, flooded, injured).

Firefighter-only

Thermal view

A thermal-camera (TIC) view as a one-tap Look — a differentiator almost no flat tool offers.

It writes itself

Debrief analytics

Every inject and decision timestamped → a decision-log timeline, an AI summary that narrates the log, and a scorecard.
